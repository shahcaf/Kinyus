# Kinyus - Premium GTA Online Services Portfolio

A modern, responsive, and high-converting portfolio website for **Kinyus**, a GTA Online services brand.

## Features
- **Modern Dark Gaming Aesthetic:** Black, dark gray, neon green, and neon purple theme.
- **Micro-Animations:** Smooth scroll reveal effects, floating hero text, and pulsing neon glows.
- **Glassmorphism:** Semi-transparent, blurred backgrounds for a premium feel.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
- **Fast Performance:** Pure HTML, CSS, and JavaScript with no heavy frameworks.

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (Intersection Observer API)
